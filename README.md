Overview

This repository contains three Java-based projects developed as part of the CodeAlpha Java Programming Internship.
The tasks focus on strengthening core Java concepts, Object-Oriented Programming (OOP), GUI development, and basic Artificial Intelligence logic.

All projects are GUI-based, interactive, and follow industry-relevant design principles.
